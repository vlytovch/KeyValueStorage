**REST SPECIFICATION:**

| Functionality        | Method | URI          | Status code |
|----------------------|--------|--------------|-------------|
| Get a pair by key    | GET    | /pairs/{key} | 200         |
| Get all pairs        | GET    | /pairs       | 200         |
| Create/Update a pair | PUT    | /pairs       | 201/200     |
| Delete a pair        | DELETE | /pairs/{key} | 200         |
